# 📖 Recipes.
> **A minimalist, version-controlled digital cookbook.**

"Recipes" is a full-stack web application designed for cooks who treat their kitchen like a lab. Instead of just overwriting a dish, this app allows you to track the evolution of a recipe through multiple versions (v1, v2, v3), ensuring you never lose that "one time it tasted perfect."

---

## ✨ Features

* **🔄 Recipe Versioning:** Save multiple iterations of the same dish. Compare v1 to v3 to see how your technique has evolved.
* **🛠️ Admin Control:** Toggle management tools (Add, Edit, Overwrite, Delete) by appending `?admin=true` to your URL.
* **🔍 Deep Search:** Find recipes by title, specific ingredients, or even keywords hidden in your "Chef's Notes."
* **🎨 Premium UI:** A clean, high-contrast interface inspired by modern editorial design.
* **⚡ Real-time Sync:** Powered by Supabase for instant data updates across all devices.

---

## 🛠️ The Tech Stack

* **Frontend:** [Next.js 15](https://nextjs.org/) (App Router)
* **Database:** [Supabase](https://supabase.com/) (PostgreSQL)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Deployment:** [Vercel](https://vercel.com/)
